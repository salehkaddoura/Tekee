Chef's Special

Chef's Special is a project I started working on with a classmate at my first hackathon Code Day. The original idea was to build a music blog from scratch but that was a little ambitious for 24hrs so we then decided on creating a robot that suggests song to a user based of their listening preferences and playlists, but also specifically to suggest aritsts that aren't mainstream or popular to help promote up and comin music. I had never used API's before and this project was a really good introduction in helping me understand how Javascript, API's, and HTML work together. The project is still being worked on and is part of a bigger project im working on Cutty Music Collective.
